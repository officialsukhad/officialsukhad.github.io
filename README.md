# SukhadOfficial - The Official Website

Welcome to the official repository of **[SukhadOfficial](https://sukhadofficial.me)**, the personal website of **Sukhad**. Here, you can find all the exciting details about my work as a **YouTuber**, **Developer**, **Designer**, and **Student**. Feel free to explore and get in touch!

## 🚀 Live Website

You can check out the live website here:  
[SukhadOfficial](https://sukhadofficial.me)

## 🔧 Features

- **Responsive Design**: Fully responsive layout, optimized for all devices.
- **Smooth Scroll**: Sleek and smooth scrolling experience.
- **Interactive Navbar**: Sticky navbar that stays at the top while scrolling.
- **Dynamic Footer**: Displays the current year dynamically using JavaScript.
- **Typing Animation**: Fun, dynamic text animation showcasing my skills.
- **Owl Carousel**: Interactive image carousel for showcasing projects.

## 💻 Technologies Used

This website is built with the following technologies:

- **HTML5**: Structuring the content of the site.
- **CSS3**: Styling the website with modern techniques.
- **JavaScript**: Adding interactivity, animations, and dynamic features.
- **jQuery**: For easier DOM manipulation and event handling.
- **Typed.js**: Implementing the typing text animation.
- **Owl Carousel**: For creating the responsive carousel of projects.
- **FontAwesome**: For beautiful icons.

## 🌟 Features & Scripts

- **Sticky Navbar**: The navbar becomes sticky when you scroll past 20px, ensuring easy access to menu items.
- **Scroll-Up Button**: A button appears after scrolling 500px, allowing users to quickly scroll back to the top.
- **Typing Animation**: Animated text that cycles through various roles like "YouTuber", "Developer", "Designer", and "Student".
- **Responsive Image Carousel**: A sleek carousel showcasing images with autoplay functionality.
